## 1.1 文本分类任务（Text Classification Tasks)
文本分类在应用中包含多个不同的任务：

1）*情感分析（Sentiment Analysis)*

分析文本数据中人们的观点（如，对产品、电影的评论），情感分类可以是二分类问题（积极、消极），也可以多分类问题（在程度上进行更细粒度的区分）。

2）*新闻分类（News Categorization)*
